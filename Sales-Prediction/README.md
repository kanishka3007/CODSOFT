# 📈 Sales Prediction 

## Overview
This project predicts product sales based on advertising budgets using a Linear Regression model. It analyzes the relationship between TV, Radio, and Newspaper advertising expenses and sales.

## Dataset
- **File:** `sales-prediction.csv`
- **Features:** TV, Radio, Newspaper
- **Target:** Sales

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

 ## How to Run
1.  Open the project in Visual Studio Code.
2. Install the required libraries.
3. Run the Python program.
      python sales_prediction.py
   
## Project Structure

Sales-Prediction/
├── sales_prediction.py/

## Output
The model predicts sales and displays the evaluation metrics, including the R² score.
