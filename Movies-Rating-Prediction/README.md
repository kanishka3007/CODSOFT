# 🎬 Movie Rating Prediction

## Objective
Build a machine learning model to predict movie ratings using movie features.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- VS Code

## Algorithm
- Linear Regression

## Features
- Duration
- Votes

## Target
- Rating

## Results
- Trained a Linear Regression model.
- Predicted movie ratings.
- Evaluated using R² Score and Mean Squared Error.
- Visualized Actual vs Predicted Ratings.
